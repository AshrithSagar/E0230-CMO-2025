\section*{Question 10}

Suppose \( A, B \) are symmetric and that the problems
\begin{equation*}
    \text{(P1)} \quad \underset{x}{\arg \min } x^{\top} A x \quad \text { and } \quad (\text{P2}) \quad \underset{x}{\arg \min } x^{\top} B x
\end{equation*}
have unique solutions \( x_{P 1}=x_{P 2}=0 \).
What is the solution to
\begin{equation*}
    \underset{x}{\arg \min } x^{\top} A B x,
\end{equation*}
and is it unique?
(hint: every symmetric PD matrix has a unique, positive definite square root -- can you prove this?)
